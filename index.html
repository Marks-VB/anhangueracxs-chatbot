Ordem correta para ligar os equipamentos

Para evitar problemas de detecção, siga esta ordem:

  1. Projetor: Ligue o projetor primeiro e aguarde ele iniciar.
  2. Computador: Com o projetor pronto, ligue o computador.

Para desligar, faça o inverso: primeiro o computador, depois o projetor.

Se o problema persistir, por favor, acione o Responsável de TI.

Projetor sem imagem (ou "Sem Sinal")

Se o projetor está ligado mas não exibe a imagem, tente os seguintes passos na ordem:

  1. Selecione a Fonte Correta: No controle remoto do projetor, pressione o botão "Source" ou "Input" e selecione a entrada correta (geralmente HDMI ou VGA).
  2. Verifique o Modo de Projeção: No teclado do computador, pressione as teclas Windows + P ao mesmo tempo. No menu que aparecer, selecione a opção "Duplicar".
  3. Reinicie o Computador: Se os passos acima não funcionarem, reiniciar o computador com o projetor já ligado pode resolver o problema.

Se nada disso resolver, o problema pode ser físico. Por segurança, não mexa nos cabos e acione o Responsável de TI para obter ajuda.

Projetor não espelha a imagem

Windows

Para que o projetor mostre a mesma imagem que a tela do notebook, faça o seguinte:

  1. Atalho do Windows: Pressione as teclas Windows + P.
  2. Escolha "Duplicar": No menu que aparecer, clique na opção "Duplicar".

Linux Mint

Para espelhar a tela no Linux Mint:

  1. Abra o Menu e procure por "Monitores".
  2. Na janela de configuração de monitores, ative a opção "Espelhar monitores".
  3. Clique em "Aplicar" para confirmar.

Se o problema persistir, por favor, acione o Responsável de TI.

Estou sem som na sala

Windows

Se o computador tem imagem mas não reproduz som, verifique os seguintes pontos:

  1. Amplificador (se houver): Verifique se o amplificador da sala está ligado. Geralmente, há um LED indicador aceso.
  2. Saída de Som do Windows: Clique no ícone de som na barra de tarefas e certifique-se de que a saída correta está selecionada (geralmente "HDMI").
  3. Volume do Sistema e da Mídia: Verifique se o volume do Windows não está no mudo. Aumente também o volume no próprio vídeo ou mídia.

Linux Mint

Para verificar o som no Linux Mint:

  1. Abra o Menu e procure por "Som".
  2. Na aba "Saída", certifique-se de que o dispositivo correto está selecionado (geralmente algo como "HDMI / DisplayPort").
  3. Verifique se o volume de saída não está no mudo.

Se o problema persistir, por favor, acione o Responsável de TI.
